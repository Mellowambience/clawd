# Animation & FX Skill

Advanced patterns for web-based animations, Three.js shaders, and kinetic visual effects.

## Core Patterns
- Kinetic Data Bursts: Particle-based data transmission along Bezier curves.
- Semantic Pulsing: Mapping emotional telemetry to color and scale frequencies.
- Glassmorphism & Depth: Layered backdrop filters and spatial tilt logic.
