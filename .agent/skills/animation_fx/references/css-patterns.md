# CSS Coding Patterns

## Glassmorphism
- `backdrop-filter: blur(24px) saturate(180%)`
- `background: rgba(10, 20, 25, 0.6)`
- `border: 1px solid rgba(255, 255, 255, 0.1)`

## Depth & Z-Index
- Background Particles: `z-index: 0`
- UI Layer: `z-index: 10`
- Overlays/Modals: `z-index: 100`
- Failsafe/Tooltips: `z-index: 200`
